## machine coding problems

### To create a new project :-

      1)  npm init
      2)  npm install --save-dev mocha    // to run test cases, install mocha library
      3)  to run test cases, add 
      
                  "scripts": {
                    "test": "mocha"
                   }
                   
          to package.json 
      4)  npm test                        // run test cases
